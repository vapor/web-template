## Getting started

If you have the Vapor Toolbox installed then simply
```
vapor new YourProject --template=https://github.com/antonyharfield/vapor-web-template
```

or
```
vapor new YourProject --web
```

to create a new project. Then run it

```
cd YourProject
vapor build
vapor run
```
